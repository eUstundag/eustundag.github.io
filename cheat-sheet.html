Ruby Enumerables

  • Any?
    • any? [{ |obj| block }] => true or false if block code is found
  • Collect
    • collect { |item| block } => new_ary (all elements which match block)
  • Combination
    • combination(n) { |c| block } => ary #[1,2,3,4].combination(2) => [[1,2],[1,3],[1,4],[2,3],[2,4],[3,4]]
  • Concat
    • abc.concat(def) # => abcdef (if they were objects)
  • Each
    • a = [ "a", "b", "c" ]
    a.each {|x| print x, " -- " }=> a -- b -- c --
  • Each_Index
    • a = [ "a", "b", "c" ] a.each_index {|x| print x, " -- " } => 0 -- 1 -- 2 --
  • Include?
    • array.include?("apple") => true or false
  • Inject
    • (5..10).inject { |sum, n| sum + n } => 45 (adds all elements in array)
  • Map
    • map { |item| block } => new_ary (all elements which match block)
  • Select
    • select { |obj| block } => array (based on expression of block being true)
  • Sort
    • a = [ "d", "a", "e", "c", "b" ]
    a.sort # => ["a", "b", "c", "d", "e"]
