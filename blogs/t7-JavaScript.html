I found from using the function and outputing the result to give me a better idea on how to utilize it. array.splice(index, deleteCount, replaceExpression) would be a little better understanding how it works.

We're doing a lot of practice with pseudo to code and I am seeing how bad pseudo or code can lead to "WTH?" situations. I have struggled with some of the content and challenges this week, it's actually more frustrating because I had almost zero issues working with Ruby. I guess javascript for me is a work in progress and I need to learn how to understand this documentation...or start my own !
